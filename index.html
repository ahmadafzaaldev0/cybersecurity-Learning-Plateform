Common Cyber Threats

Phishing

Deceptive attempts to steal sensitive information by pretending to be a trustworthy entity.

Malware

Malicious software designed to harm or exploit any device, service, or network.

Weak Passwords

Easily guessable or reused passwords that compromise account security.

Social Engineering

Psychological manipulation to trick individuals into revealing confidential information.

Ransomware

Malicious software that encrypts files and demands payment for their release.

Man-in-the-Middle

Attackers secretly intercept and relay communications between two parties.

AI-Powered Attacks

Cybercriminals use artificial intelligence to automate and enhance their attack methods.

Cloud Vulnerabilities

Misconfigured cloud services and APIs that expose sensitive data to unauthorized access.

Mobile Malware

Malicious software specifically designed to target mobile devices and apps.

DDoS Attacks

Overwhelming a target system with traffic from multiple sources to disrupt service.

Business Email Compromise

Impersonating executives to trick employees into transferring money or sensitive data.

Safety Tips

Use Strong Passwords

Create complex, unique passwords for each account and use a password manager.

Enable 2FA/MFA

Add an extra layer of security with multi-factor authentication.

Update Regularly

Keep your operating system, software, and apps up to date with the latest security patches.

Verify Links & Attachments

Always verify the sender and hover over links before clicking. Don't open suspicious attachments.

Use Secure Networks

Avoid using public Wi-Fi for sensitive transactions. Use a VPN when necessary.

Backup Data

Regularly back up important data using the 3-2-1 rule: 3 copies, 2 different media, 1 offsite.

Be Wary of Social Engineering

Verify requests for sensitive information, even if they appear to come from a trusted source.

Secure Mobile Devices

Use biometrics, strong passcodes, and keep your mobile OS and apps updated.

Secure Cloud Storage

Enable encryption, use strong authentication, and regularly review sharing permissions for cloud services.

Secure Home Network

Change default router credentials, enable WPA3 encryption, and create a separate guest network.

Monitor Financial Accounts

Regularly check bank and credit card statements for unauthorized transactions.

Use Antivirus Software

Install and maintain reputable antivirus and anti-malware solutions on all devices.

Enable Security Alerts

Set up alerts for suspicious activities on your financial and online accounts.

Cybersecurity Glossary

Firewall

A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

VPN (Virtual Private Network)

Creates a secure, encrypted connection over a less secure network, such as the public internet.

Encryption

The process of converting data into a code to prevent unauthorized access.

Zero-Day Exploit

A cyber attack that occurs on the same day a weakness is discovered in software.

Phishing

Fraudulent attempts to obtain sensitive information by disguising as a trustworthy entity.

Malware

Malicious software designed to harm, exploit, or damage computer systems.

Ransomware

Malware that encrypts files and demands payment for their release.

DDoS (Distributed Denial of Service)

Overwhelming a target system with traffic from multiple sources to disrupt service.

2FA/MFA

Two-Factor/Multi-Factor Authentication adds an extra layer of security beyond just a password.

Social Engineering

Psychological manipulation to trick users into making security mistakes or giving away sensitive information.

Penetration Testing

Authorized simulated cyber attacks to evaluate system security.

Dark Web

Part of the internet that isn't indexed by search engines, often used for illegal activities.

Endpoint Security

Protecting network endpoints like computers and mobile devices from malicious activity.

SSL/TLS

Cryptographic protocols designed to provide communications security over a computer network.

Zero Trust

Security model that requires all users to be authenticated and authorized before accessing resources.

IoT Security

Protecting internet-connected devices and networks from cyber threats.
